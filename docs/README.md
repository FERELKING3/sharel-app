Glossaire

- peer : un appareil participant au transfert (sender ou receiver).
- host : l'appareil qui expose un point HTTP et sert les fichiers.
- client : l'appareil qui se connecte au host pour télécharger les fichiers.
- sessionId : identifiant de session unique côté host.
- transferId : identifiant unique d'un transfert (corrélé aux logs pour observabilité).
