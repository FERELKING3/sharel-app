# Routing (GoRouter)

Toutes les routes sont centralisées dans `lib/core/router/app_router.dart`.

Principes:

- Utilisation de `ShellRoute` pour l'`AppShell` (bottom navigation).
- Routes principales:
  - `/` : `HomePage`
  - `/sender` : page d'envoi
  - `/transfer/preparation`, `/transfer/host`, `/transfer/join`, `/transfer/client` : flux de transfert
  - `/receiver` : page réception

Notes:

- Lorsqu'une page doit s'ouvrir en dehors du shell (ex: dialogs, modals), la route est déclarée avec `parentNavigatorKey: _rootNavigatorKey`.
